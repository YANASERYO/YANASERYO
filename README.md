## 制作中のアプリ

### shrineprayservice
神社向けの祈願予約・業務管理システムを開発中です。

主な機能：

- 祈願予約フォーム
- 管理者・スタッフログイン
- 予約一覧・編集・削除
- 祈願済みステータス管理
- 郵便番号データを用いた住所補助機能
